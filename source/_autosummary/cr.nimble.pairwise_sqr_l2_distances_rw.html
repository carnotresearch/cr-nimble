cr.nimble.pairwise_sqr_l2_distances_rw¶

cr.nimble.pairwise_sqr_l2_distances_rw(A, B)[source]¶

Computes the pairwise squared distances between points in A and points in B where each point is a row vector

Parameters

  • A (jax.numpy.ndarray) – A set of M K-dimensional points (row-wise)

  • B (jax.numpy.ndarray) – A set of N K-dimensional points (row-wise)

Returns

An MxN matrix D of squared distances between points in A and points in B

Return type

(jax.numpy.ndarray)

  • Let the ambient space of points be \(\mathbb{F}^K\).

  • \(A\) contains the points \(a_i\) with \(1 \leq i \leq M\) and each point maps to a row of \(A\).

  • \(B\) contains the points \(b_j\) with \(1 \leq j \leq N\) and each point maps to a row of \(B\).

Then the distance matrix \(D\) is of size \(M \times N\) and consists of:

(1)¶\[d_{i, j} = \| a_i - b_j \|_2^2 = \langle a_i - b_j , a_i - b_j \rangle\]
