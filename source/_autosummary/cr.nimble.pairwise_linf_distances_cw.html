cr.nimble.pairwise_linf_distances_cw¶

cr.nimble.pairwise_linf_distances_cw(A, B)[source]¶

Computes the pairwise Chebyshev distances between points in A and points in B where each point is a column vector

Parameters

  • A (jax.numpy.ndarray) – A set of M K-dimensional points (column-wise)

  • B (jax.numpy.ndarray) – A set of N K-dimensional points (column-wise)

Returns

An MxN matrix D of Chebyshev distances between points in A and points in B

Return type

(jax.numpy.ndarray)
